Si une expérience est tournée en island model alors un fichier connections_send.txt est généré. Si plusieurs fichiers sont générés (cela dépend de votre expérience) alors il faut concaténer les fichiers par un script simple. Le fichier doit ensuite être placé dans GridVis/GridVis/data.

Ouvrer le projet Gridvis, avec eclipse par exemple. Il faut indiquer le fichier que vous voulez visualiser dans la classe GridVis. Vous pouvez ensuite lancer le programme.

Sur l'interface, le boutton "help" est disponible pour toute question.
